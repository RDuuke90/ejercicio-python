### Ejercicio 1: Figura Geométrica
#### Entrada:
- Crear un Circulo con radio 5 y un Cuadrado con lado 4.
- Llamar a calcular_area() y calcular_perimetro().
### Salida esperada:
- **Circulo.calcular_area() retorna**: 78.54
- **Circulo.calcular_perimetro() retorna**: 31.42
- **Cuadrado.calcular_area() retorna**: 16
- **Cuadrado.calcular_perimetro() retorna**: 16
